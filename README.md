# E-commerce Sales Intelligence Dashboard

## Overview

This project analyses e-commerce marketplace data to understand revenue trends, customer behaviour, delivery performance, and seller impact using SQL, Python, and Tableau.

## Tools Used

* Python (data cleaning & transformation)
* SQL (data analysis)
* Tableau (dashboard & visualisation)

## Key Insights

* Late deliveries significantly reduce customer satisfaction
* Revenue shows clear seasonal trends and growth patterns
* A small number of categories drive a large share of revenue
* Seller performance impacts delivery and customer experience
